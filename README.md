# Protheus-API
A API served by SQL protheus database to make reports and KPIs. Using React.js and Node.js

# Script for update the app on Server

1º - Login with SSH<br/>
2º - cd Protheus-API<br/>
3º - git pull<br/>
4º - cd frontend<br/>
5º - npm run build<br/>
6º - pm2 restart all<br/>
